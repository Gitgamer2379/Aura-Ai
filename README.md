## 🇧🇷 Português

A **Aura AI** é uma interface de chat moderna e elegante que utiliza a inteligência artificial do Google Gemini para oferecer conversas fluidas em diversos idiomas. O projeto foca em **privacidade e segurança**, permitindo que o usuário utilize sua própria API Key de forma privada no navegador.

### 🌟 Funcionalidades
- **Poliglota:** Responde automaticamente no idioma em que é consultada.
- **Design "Aura":** Interface escura (Dark Mode) com gradientes e animações suaves.
- **Segurança:** A chave de API não é exposta no código; é salva apenas no `localStorage` do usuário.

### 🛠️ Tecnologias
- HTML5 / CSS3 / JavaScript (ES6+)
- Google Gemini API (Modelo 1.5 Flash)

## 🇺🇸 English

**Aura AI** is a modern and elegant chat interface powered by Google Gemini AI, designed to provide seamless conversations across multiple languages. The project prioritizes **privacy and security**, enabling users to securely use their own API Key within the browser.

### 🌟 Features
- **Polyglot:** Automatically responds in the language it is addressed in.
- **"Aura" Design:** Dark Mode interface featuring modern gradients and smooth animations.
- **Security:** The API key is never exposed in the source code; it is stored only in the user's `localStorage`.

### 🛠️ Technologies
- HTML5 / CSS3 / JavaScript (ES6+)
- Google Gemini API (1.5 Flash Model)

## 🇪🇸 Español

**Aura AI** es una interfaz de chat moderna y elegante que utiliza la inteligencia artificial de Google Gemini para ofrecer conversaciones fluidas en diversos idiomas. El proyecto se centra en la **privacidad y seguridad**, permitiendo que el usuario utilice su propia API Key de forma segura en el navegador.
