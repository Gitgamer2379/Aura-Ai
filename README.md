# ✨ Aura AI - Assistente Multilíngue

A **Aura AI** é uma interface de chat moderna e elegante que utiliza a inteligência artificial do Google Gemini para oferecer conversas fluidas em diversos idiomas. 

O projeto foi construído focando em **privacidade** e **segurança**, permitindo que o usuário utilize sua própria API Key de forma local no navegador.

## 🚀 Demonstração
Você pode testar a Aura AI clicando aqui: [COLE O LINK DO SEU GITHUB PAGES AQUI]

## 🌟 Funcionalidades
- **Poliglota:** Entende e responde em dezenas de idiomas automaticamente.
- **Design "Aura":** Interface escura (Dark Mode) com gradientes modernos e animações suaves.
- **Segurança (Client-Side):** A chave de API não é exposta no código; ela é salva apenas no `localStorage` do navegador do usuário.
- **Responsiva:** Funciona bem em computadores e celulares.

## 🛠️ Tecnologias Utilizadas
* **HTML5** - Estrutura da aplicação.
* **CSS3** - Design personalizado com Flexbox e Gradientes.
* **JavaScript (ES6+)** - Lógica de comunicação com a API e manipulação do DOM.
* **Google Gemini API** - O "cérebro" por trás das respostas.

## 🔑 Como usar
1. Obtenha sua chave de API gratuita no [Google AI Studio](https://aistudio.google.com/).
2. Ao abrir a **Aura AI**, insira sua chave quando solicitado.
3. Se precisar trocar a chave, utilize o botão "Alterar Chave" no cabeçalho.
